# Ved Singh — Personal Portfolio

A personal portfolio website for **Ved Singh**, a final year MBBS student at [Dr. D. Y. Patil Medical College, Pune](https://dypatil.edu/pune/).

## About Ved

- 🩺 **Final Year MBBS Student**
- ✈️ Solo Traveller
- 📷 Photographer
- 🍳 Cooking Enthusiast

## Tech Stack

- [Next.js 16](https://nextjs.org/) with App Router
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)

## Sections

- **Hero** — Full-screen intro with background image
- **About** — Bio, college info, and stats
- **Interests** — Solo Travelling, Photography, Cooking
- **Gallery** — Masonry photo grid with lightbox viewer
- **Connect** — Links to Instagram (@vedawsm & @food_ved_)

## Get Started

```bash
# Install dependencies
bun install

# Run dev server
bun run dev

# Build for production
bun run build
```
